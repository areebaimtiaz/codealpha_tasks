# Task 3: Data Visualization

##  Overview
This project focuses on converting analyzed data into meaningful visual representations to identify patterns and communicate insights effectively.

##  Objective
- Visualize author distribution
- Create insightful charts and graphs
- Improve data storytelling
- Understand patterns visually

##  Tools Used
- Python
- Pandas
- Matplotlib

##  Dataset
- Source: all_quotes.csv (from Task 1)

##  Visualizations Created

### 1. Bar Chart
- Top 10 most quoted authors

### 2. Histogram
- Distribution of quotes per author

### 3. Pie Chart
- Share of top authors in dataset

## Key Insights
- A small number of authors dominate the dataset
- Albert Einstein has the highest number of quotes
- Most authors appear only a few times
- Data is unevenly distributed

##  Key Learnings
- Data visualization techniques using Matplotlib
- Choosing correct chart types
- Converting data into visual insights
- Storytelling through graphs
