
# Task 4: Sentiment Analysis

##  Overview
This project applies Natural Language Processing (NLP) techniques to analyze the emotional tone of quotes and classify them into sentiment categories.

##  Objective
- Perform sentiment analysis on text data
- Classify quotes as Positive, Negative, or Neutral
- Understand emotional patterns in text
- Apply NLP techniques using Python

##  Tools Used
- Python
- TextBlob
- Pandas
- Matplotlib
- 
##  Dataset
- Source: all_quotes.csv (from Task 1)

## Methodology

### 1. Text Processing
- Each quote analyzed using TextBlob NLP library

### 2. Sentiment Classification
- Polarity > 0 → Positive
- Polarity < 0 → Negative
- Polarity = 0 → Neutral

### 3. Feature Engineering
- Added new column: Sentiment

##  Results
- Positive quotes are dominant in dataset
- Neutral quotes represent factual statements
- Negative quotes are minimal

- Bar chart of sentiment distribution
- Pie chart of sentiment percentages
